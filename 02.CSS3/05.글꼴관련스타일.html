font-size : 글꼴의 크기를 변경할 때 사용

        선택자 {font-size:크기(px|em|rem|%)}
        - px : 픽셀단위로 고정크기
        - em : 부모요소의 글꼴크기의 배수(1.5em)
        - rem : HTML태그의 폰트 크기의 비율로 계산 (배수 1.5rem) (이걸 많이씀)
        - % : 상위요소 폰트크기의 비율로 계산(150%)

        - vw(가로), vh(세로) : 현재 보이는 창의 크기에 따라 값이 변한다.(창의 width:1000px -> 1vw : 10px) (반응형할때 많이씀)


        옵션들
        * 고정값
        xx-small : 10px
        x-small : 10px
        small : 13px
        medium : 16px
        large : 18px
        x-large : 24px
        xxlarge : 32px

        * 상대값 (부모기준)
        larger : 부모의 글자크기에 120%
        smaller : 부모의 글자크기에 1/6

  • 가변크기 rem테스트
  • 가변크기 %테스트

font-weight : 글꼴의 두께를 변경할 때 사용

        선택자{
            font-weight:normal|bold|bolder|lighter|100|400|700
        }
  • 원래굵기
  • 굵은 글꼴로 변경
  • 원래 굵기보다 더 굵게
  • 원래 굵기보다 더 가늘게

font-variant : 영문텍스트 문구를 작은 대문자로 변경할 때 사용

    선택자{
        font-variant:small-caps;
    }
