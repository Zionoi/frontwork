그 밖의 글자를 다루는 태그들

일반글꼴 (태그로 감싸지 않는 텍스트)

b : 글자를 굵게 표시하는 태그

strong : 글자를 굵게 표시하는 태그 + 스크린리더(시각장애인용 리더에도 적용가능 좀 더 강하게 읽어줌)

i : 글자를 기울여서 표시하는 태그

em(emphasis) : 특히 강조할 때 글자를 기울여서 표시하는 태그(스크린리더가 강조해서 읽어줌)

mark : 형광펜 효과를 내는 태그

u : 글자에 밑줄이 그어지는 태그

s : 글자에 취소선을 넣어주는 태그

small : 글자를 작게 표현해 주는 태그

기본글자에 sub : 아랫첨자를 나타내는 태그
sup : 위첨자를 나타내는 태그

abbr : 줄임말 표시
HTML란 월드 와이드 웸(World Wide Web)의 핵심 마크업 언어이다.

blockquote : 인용문
